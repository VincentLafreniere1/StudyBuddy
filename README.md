# StudyBuddy
StudyBuddy will be an engineering study agent that can summarize pdfs, detect formulas and list them in another pdf with explanation and usage context.

I will divide the work into a few bullet points and will keep adding features moving foward.

1. Needs to be able to reliably read pdfs and extract text from it (x)
2. Needs to include an "AI Agent" that can summarize the pdf ()
3. Needs to use ML algorithms to detect a formula in the extracted text ()
4. ...
   
The core idea is to have a trained agent that actually helps you understand lectures and academic papers in a reliable way
